
## JuanLauncher 🚀

 Bienvenido al repositorio oficial de JuanLauncher, un proyecto personal diseñado para Probar y Descargar Juegos para PC y Android

    Enlace: https://juancho033.github.io/juanlauncher/

## 🚀 Instalación y Uso
Si estás aquí porque el proyecto ha ganado popularidad y quieres revisar el código, ¡eres bienvenido! Para ejecutarlo localmente:

 1: Clona este repositorio:

```bash
  https://github.com/juancho033/juanlauncher.git
```
2: Abre el archivo index.html en tu navegador preferido o usa una extensión como Live Server en VS Code.
